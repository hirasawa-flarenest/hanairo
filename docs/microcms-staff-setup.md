# microCMS スタッフ紹介 設定方法

## API設定

### 1. 新規API作成
- API名: `スタッフ紹介`
- エンドポイント: `staff`
- APIの型: `リストAPI`

### 2. APIスキーマ設定

以下のフィールドを設定してください:

#### name (名前)
- フィールドID: `name`
- 表示名: `名前`
- 種類: `テキストフィールド`
- 必須: `ON`

#### position (役職)
- フィールドID: `position`
- 表示名: `役職`
- 種類: `テキストフィールド`
- 必須: `OFF`
- 説明: `施設長、副施設長、スタッフなど`

#### photo (プロフィール写真)
- フィールドID: `photo`
- 表示名: `プロフィール写真`
- 種類: `画像`
- 必須: `OFF`
- 推奨サイズ: `400x400px 以上の正方形`
- 説明: `Retinaディスプレイ対応のため、実際の表示サイズ（200x200px）の2倍以上を推奨します`

#### bio (紹介文)
- フィールドID: `bio`
- 表示名: `紹介文`
- 種類: `テキストエリア`
- 必須: `OFF`
- 説明: `スタッフの経歴や趣味、メッセージなど`

#### displayOrder (表示順)
- フィールドID: `displayOrder`
- 表示名: `表示順`
- 種類: `数値`
- 必須: `ON`
- 説明: `小さい数字ほど上位に表示されます`

### 3. 公開状態の管理

スタッフの表示/非表示は、microCMSの標準公開機能を使用します：
- **公開**: サイトに表示されます
- **下書き**: サイトに表示されません

コンテンツを非表示にしたい場合は、下書き状態に戻してください。

## APIプレビュー設定（オプション）

Next.jsのプレビュー機能を使用する場合:
- プレビューURL: `https://your-domain.com/api/preview?slug=/staff&draftKey={DRAFT_KEY}`

```json
{
  "name": "山田 太郎",
  "position": "施設長",
  "photo": {
    "url": "https://images.microcms-assets.io/...",
    "width": 400,
    "height": 400
  },
  "bio": "地域の子育て支援に10年以上携わってきました。子どもたちの笑顔と保護者の皆さまの安心のために、スタッフ一同頑張っています。",

```
